Markdown Syntax

Apr 12th, 2018 1:51 pm

As u know octopress use markdown as a result file, and for that there is different syntax from html basic. So here example syntax for writting in markdown.

==HEADER==

Result:

This is H1

This is H6

…

==LISTS==

Unordered

Result:

  • List Unordered 1
  • List Unordered 2
  • List Unordered 2a
  • List Unordered 2b

Ordered

Result:

  1. List Ordered 1
  2. List Ordered 2
  3. List Ordered 3
  4. List Ordered 3a
  5. List Ordered 3b

…

==IMAGES==

Result:

As of this writing, Markdown has no syntax for specifying the dimensions of an image; if this is important to you, you can simply use regular HTML <img> tags.

…

==Links==

Result:

http://google.com - automatic! Google

…

==BLOCKQUOTES==

Result:

As Grace Hopper said:

I’ve always been more interested in the future than in the past.

…

==BACKSLASH ESCAPES==

Markdown allows you to use backslash escapes to generate literal characters which would otherwise have special meaning in Markdown’s formating syntax.
